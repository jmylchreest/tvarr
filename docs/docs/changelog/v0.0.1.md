# v0.0.1

**Released:** 2024-01-01

Initial release of tvarr.

## Added

- Stream source management (M3U, Xtream)
- EPG source management (XMLTV, Xtream)
- Proxy playlist generation
- Expression-based filtering
- Data mapping rules
- Relay mode with transcoding
- Hardware acceleration (VAAPI, NVENC, QSV, AMF)
- Distributed transcoding with ffmpegd workers
- Client detection rules
- Web UI with dark mode
- Docker and Kubernetes deployment
