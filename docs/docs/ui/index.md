# User Interface

tvarr provides a modern web interface for managing your streams.

## Navigation

The sidebar provides access to all sections:

- **Dashboard** - Overview and quick actions
- **Sources** - Stream and EPG sources
- **Channels** - Browse ingested channels
- **Proxies** - Output playlist configuration
- **EPG** - Electronic Program Guide viewer
- **Admin** - Filters, rules, encoding, and more
- **Settings** - Global configuration
- **Transcoders** - Worker status (if using distributed mode)
