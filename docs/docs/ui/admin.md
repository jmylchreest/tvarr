# Admin

The Admin section contains advanced configuration options.

## Filters

**Admin > Filters**

Create include/exclude rules for channels and EPG. See [Filters](/docs/next/rules/filters).

## Data Mapping

**Admin > Data Mapping**

Create transformation rules to modify channel metadata. See [Data Mapping](/docs/next/rules/data-mapping).

## Encoding Profiles

**Admin > Encoding Profiles**

Configure video/audio encoding settings for transcoding. See [Encoding Profiles](/docs/next/transcoding/encoding-profiles).

## Client Detection

**Admin > Client Detection**

Create rules to serve different quality based on device. See [Client Detection](/docs/next/rules/client-detection).

## Encoder Overrides

**Admin > Encoder Overrides**

Custom FFmpeg encoder settings for specific scenarios:

- Override encoder selection
- Custom encoder options
- Per-codec settings

## Logos

**Admin > Logos**

Manage the logo cache:

- View cached logos
- Clear cache
- Configure retention

## Backups

**Admin > Backups**

Export and import configuration:

- **Export** - Download all rules, sources, proxies
- **Import** - Restore from a backup
- **Schedule** - Automatic backups
