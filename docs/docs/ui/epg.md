# EPG

The EPG page provides a visual program guide.

## Guide View

A grid showing channels and their programs:

- **Rows** - Channels
- **Columns** - Time slots
- **Cells** - Programs

Click a program to see details.

## Navigation

- **Date picker** - Jump to a specific date
- **Now button** - Jump to current time
- **Scroll** - Pan through the guide

## Filtering

Filter the guide by:

- Search for specific channels
- Filter by group/category
- Filter by proxy

## Program Details

Click a program to see:

- Title and description
- Start/end times
- Category
- Episode/season info (if available)
