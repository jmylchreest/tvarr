# v0.0.4

**Released:** 2025-12-15

## Changed

- Configurable pnpm version in CI (defaults to latest)
- Container runtime now configurable via environment variable

## Fixed

- Docker build permissions for GitHub Container Registry
- Provenance settings in FFmpeg Docker build

## Removed

- ARM64 Docker builds (temporarily, pending FFmpeg base image support)
