# v0.0.1

**Released:** 2025-12-14

Initial release of tvarr.

## Added

- Stream source management (M3U, Xtream Codes)
- EPG source management (XMLTV, Xtream)
- Proxy playlist generation with merged channels
- Expression-based channel filtering
- Data mapping rules for metadata transformation
- Relay mode with real-time transcoding
- Hardware acceleration support (VAAPI, NVENC, QSV, AMF)
- Distributed transcoding with ffmpegd workers
- Client detection rules for device-specific profiles
- E-AC3 audio demux and muxing support
- Scheduled job catch-up for missed runs
- Web UI with dark mode support
- Docker and Kubernetes deployment options
