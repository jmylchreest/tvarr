# Dashboard

The dashboard shows your system status at a glance.

## Status Cards

- **Sources** - Count of stream and EPG sources
- **Channels** - Total ingested channels
- **Proxies** - Active proxy playlists
- **Jobs** - Running and queued jobs

## Quick Actions

Common tasks accessible from the dashboard:

- **Ingest All** - Trigger ingestion for all sources
- **Generate All** - Regenerate all proxy playlists

## Recent Activity

Shows recent job completions and any errors.
