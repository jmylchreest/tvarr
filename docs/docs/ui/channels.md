# Channels

The Channels page shows all channels ingested from your sources.

## Browsing

### Search

Type in the search box to filter by channel name.

### Filters

Filter channels by:

- **Source** - Show channels from specific sources
- **Group** - Show channels in specific categories

### Sorting

Click column headers to sort by:

- Name
- Group
- Channel Number
- Source

## Channel Details

Click a channel to see:

- Full metadata (tvg-id, logo, group)
- Stream URL
- EPG data (if matched)

## Bulk Actions

Select multiple channels for bulk operations:

- Export selected
- View in EPG
